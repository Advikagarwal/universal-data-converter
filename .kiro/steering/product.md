# Universal Data Format Converter

A browser-based single-page application for converting data between JSON, YAML, XML, and CSV formats with intelligent syntax repair and schema generation capabilities.

## Core Purpose

Enable seamless conversion between multiple data formats with advanced features like auto-detection, syntax repair, and schema generation - all processed locally in the browser for privacy.

## Key Features

- Multi-format conversion (JSON ↔ YAML ↔ XML ↔ CSV)
- Smart auto-detection with confidence scoring
- Intelligent syntax repair with preview
- Schema generation (TypeScript interfaces, JSON schemas)
- Pretty printing and minification
- CSV intelligence (type detection, header recognition, custom delimiters)
- Real-time validation with precise error reporting
- Privacy-first: all processing happens client-side

## Target Users

Developers and data professionals who need to quickly convert between data formats without uploading sensitive data to external services.
